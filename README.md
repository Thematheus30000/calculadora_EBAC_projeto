# Python Calculator Project

This project is a simple calculator developed in Python, along with a shell script to execute it in a Linux environment.

## Files

- `calculadora_teste.py`: Main Python calculator code
- `calculadora.sh`: Shell script to run the calculator
- `README.md`: Project documentation

## How to run the .sh file

1. Open the terminal in the project folder
2. Give execution permission:

chmod +x calculadora.sh

3. Run the script:

./calculadora.sh

## Python Code Explanation

The Python script performs a simple calculator operation.

It:

- asks for the user's name
- requests two numbers
- asks which mathematical operation to perform
- executes the operation using conditional logic (`if`, `elif`, `else`)
- displays the result in the terminal

Supported operations:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Run Python directly

You can also run the calculator without the shell script:

python3 calculadora_teste.py
